# Store Management System

A **full-stack web application** designed to streamline **inventory tracking, material requests, approval workflows, and transaction logging** within an organization.

Built using **React, Node.js, and MySQL**, the system focuses on **usability, transparency, and operational efficiency**.

---

# 🚀 Features

## 🔹 Inventory Management

- Add, update, and track stock levels
- Real-time visibility of available and requested items
- Monitor inventory movement across the system

---

## 🔹 Request & Approval Workflow

- Raise material requests
- Manager/Admin approval system
- Track the status of each request

---

## 🔹 Role-Based Authentication

- Secure login system
- Role-based access for **Admin, Manager, and Staff**
- Prevents unauthorized access to sensitive data

---

## 🔹 Real-Time Reporting

- Stock insights and usage patterns
- Dashboard summaries for quick decision-making
- Improved operational visibility

---

## 🔹 Transaction Logging

- Complete audit trail of inventory movement
- Ensures transparency and accountability
- Helps maintain accurate records

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```
git clone <repo-url>
cd store-management-system
```

---

## 2️⃣ Install Dependencies

### Frontend

```
cd client
npm install
```

### Backend

```
cd server
npm install
```

---

## 3️⃣ Configure Database

- Import the **MySQL dump** located in the `/database` folder.
- Update database credentials in:

```
server/config.js
```

---

## 4️⃣ Start the Project

### Start Backend

```
cd server
npm run dev
```

### Start Frontend

```
cd client
npm start
```

The application will now run locally.
